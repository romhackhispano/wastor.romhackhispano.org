Ninja Cop

Consola: Gameboy Advance

  

Las características que puedes disfrutar en este proyecto son:

- Salvo error u omisión, todo en español.
- Incluídos todos los caracteres propios del español.
- Traducción de todos los gráficos, menos los créditos finales y alguna cosilla ambiental, que considero rompe la ambientación traducirla.
